# TikTok Голосование — Donat Drag Race (Web UI)

Статический проект одной страницы под 9:16.

## Запуск
Открой index.html в браузере.

## Донаты
В app.js вызывай:
  donateToLane(laneIdx, amount)
когда приходит событие доната из TikTok.

Сейчас есть demoDonate() для теста (его можно удалить).

## Быстрая настройка
- LANES: 4 или 6
- ROUND_SECONDS: длительность раунда
