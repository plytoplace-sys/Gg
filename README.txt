STREAM NEON SVG (XML) v2.1

Если у тебя было "как будто ничего нет" — это почти всегда WebView/кеш.
В этой версии:
- нет <use xlink:href> (часто ломается в webview) — машины клонируются через cloneNode из SVG-шаблона
- свечение дороги/фон сделаны ВИДИМЫМИ (не "в полтона")
- город и неоновые вывески заметнее

Запуск:
- лучше через GitHub Pages или любой http сервер (чтобы не было ограничений file://)
- на телефоне открывай в Chrome/браузере, а не в "просмотрщике файлов".

Важно про TikTok:
Зрители не кликают по твоему видео-оверлею. Кликабельность — только на веб-странице по ссылке (в профиле/описании/QR).
