# Donat Drag Race — NEON SVG (TOP)

Без картинок: весь арт — SVG (XML).

Есть:
- 6 машин
- у каждой 3 тест‑кнопки (уникальные лайки) => BOOST / BURST / NITRO
- разные ускорения
- неоновые следы
- вспышка при донате
- 8‑bit звук (WebAudio)
- комментарии доната (лента)
- ТОП донатеров (сверху)
- общие победы (localStorage)
- авто‑следующий раунд после победы

Запуск: открой index.html (или GitHub Pages).
