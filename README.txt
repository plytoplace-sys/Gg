STREAM NEON SVG (XML)

Это версия под "стрим-оверлей":
- Всё в SVG (XML), без картинок.
- Глубина/перспектива, свечение трека, город на фоне.
- При донате: звук 8-bit + вспышка + огонь/пиксели + pop-текст над машиной + запись в ленту.
- Сверху: полоса топ-донатеров (аватар-инитиалы), мини-кнопка "+" (демо) и кнопка ↗ (открывает профиль в браузере).

ВАЖНО: зрители TikTok в видео НЕ могут кликать по элементам твоего оверлея.
Кликабельность возможна:
- если это открыто у тебя в браузере (OBС Browser Source) — ты можешь кликать,
- либо если ты даёшь ссылку/QR на страницу — зритель кликает по ссылке в описании/профиле и уже там нажимает.

Запуск: открой index.html или задеплой на GitHub Pages.
