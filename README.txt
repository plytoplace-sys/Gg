Donat Drag Race — STREAM ULTRA (SVG/XML)

Это УЛЬТРА-версия под OBS Browser Source (мощный ПК):
- Детализированные SVG "модели" машин (слои: body/metal/clearcoat/glass/lights/underglow/panels)
- Город с мерцающими окнами (SVG animate)
- Дорога с движущимися неоновыми streaks
- При донате: 8-bit звук + pop-текст + огонь + пиксели
- При NITRO: волна по центру + camera shake

Запуск:
1) Открой index.html, или лучше запусти локально через http (например, Live Server в VSCode).
2) В OBS: Add -> Browser Source -> укажи URL (локальный/Pages).

Важно:
- Зрители TikTok НЕ могут кликать по элементам видео-оверлея.
  Кнопка ↗ в топ-баре кликабельна только если это открыто как web-страница (или ты кликаешь в OBS).
