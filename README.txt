# Donat Drag Race v2

- Canvas рисует трассу с перспективой (сужение к финишу)
- Машинки двигаются по своим маршрутам (слегка разные)
- Машинки уменьшаются к финишу
- Цветные следы от старта к текущей позиции
- Победитель объявляется при пересечении финиша

## Запуск
Открой index.html

## Подключение донатов
В app.js вызывай donateToLane(laneIdx, amount) при событии доната.
